# rangine
Game engine library by me.

Aiming to be a single header stb-style library.

## How to use
Include header in project:
```C
#include <rangine.h>
#include "rangine.h"
```

## Contribute
Feel free to make contributions! Try to fit in with the current codebase which may change.
