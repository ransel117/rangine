# rangine
Game engine by me.

Aiming to be a single header stb-style library.

## How to use
Include header in project:
```
#include <rangine.h>
```
or
```
#include "rangine.h"
```

## Contribute
Feel free to make contributions! Try to fit in with the current code base which may change.
